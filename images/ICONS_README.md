# PWA Icons Guide

Your Food Explorer app needs icons in these sizes. You can create them using:

## Option 1: Online Icon Generator (Easiest)
1. Visit: https://favicon.io/favicon-generator/

## Option 2: Use Canva (Professional)
1. Go to canva.com
2. Create custom size: 512x512
3. Design your restaurant icon
4. Export as PNG
5. Use https://realfavicongenerator.net/ to generate all sizes

## Option 3: Quick Placeholder (For Testing)
Use emojis or simple designs and resize:
- 72x72, 96x96, 128x128, 144x144, 152x152, 192x192, 384x384, 512x512

## Required Files:
Place these in `images/` folder:
- icon-72x72.png
- icon-96x96.png
- icon-128x128.png
- icon-144x144.png
- icon-152x152.png
- icon-192x192.png
- icon-384x384.png
- icon-512x512.png
